# Allen Ovoderoye — Portfolio

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Vercel auto-detects Create React App — just click Deploy

## Local Development

```bash
npm install
npm start
```

## Before deploying

- Update your real email in `App.js` (search `allen@example.com`)  
- Update your real GitHub and LinkedIn URLs (search `https://github.com` and `https://linkedin.com`)
- Add your phone number to the Contact section if desired
